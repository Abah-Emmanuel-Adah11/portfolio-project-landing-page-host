About

This project was inspired by the challenge of creating a 3D maze game using raycasting. It combines elements of creativity, problem-solving, and technical skill, making it an exciting endeavor to undertake.

As a Portfolio Project for the ALX SWE Program, this project allowed us to showcase our abilities in game development, C programming, and software engineering. It was a collaborative effort between Mr. Emmanuel, A. and Mr Victor, G. requiring effective communication and teamwork.

Overall, the Maze Game project represents our dedication to learning and our passion for creating immersive experiences through technology.
